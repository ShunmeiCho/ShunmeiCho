## Research Focus

<table>
<tr>
<td width="50%" valign="top">

### Hierarchical Reasoning Models

**Small-Data Inference under Constraints**

- Compositional generalization (key-value binding, multi-step chains)
- Robust inference under partial observability
- Distribution shift & shortcut detection

**Methodology:**
Config-driven experiments | Clear baselines | Honest failure analysis | W&B tracking

</td>
<td width="50%" valign="top">

### LLM x Robotics

**Unitree GO2 End-to-End Prototype**

```
Voice -> ASR -> LLM Reasoning
  -> Action Command -> ROS2/SDK
    -> Execution -> Feedback Loop
```

**Key Concerns:**
Interface contracts | Latency budgets | Safety filters | Fallback behaviors

</td>
</tr>
